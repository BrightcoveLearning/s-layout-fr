Pages are used to display video content in a Brightcove Beacon application. To create a page, follow these steps:

  1. Open the Layout module.
  2. At the top of the page list click Create Page. The Creating Page panel opens.
  3. Enter a Page Name. Note that the character limit is beyond the logical limit for a page name.
  4. Click a Layout to select it. The Layout controls how the content is displayed on the page. Selecting a layout updates the page preview on the right.
  5. Click Create.
  6. Click Add playlist. The Choose Playlist panel opens displaying a list of created playlists.
  7. Select one or more playlists to add to the page by hovering over the playlist and then clicking Add. As playlists are added, they appear on the page preview. Note that some layouts only accept one playlist.
  8. Click X to close playlists panel.
  9. To make the new page visible in your apps, click Publish at the top of the navigation panel.
  10. Click Back to All Pages to return to the list of pages.

The page that was created appears in the list of pages. To edit any of the page details, click on the page name.
