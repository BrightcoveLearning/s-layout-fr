Les paramètres de disponibilité des pages peuvent être utilisés pour contrôler:

  • Les dates auxquelles une page est disponible
  • Le ou les appareils sur lesquels une page est disponible pour être visualisée
  • Les emplacements où une page peut être consultée

Pour configurer les paramètres de disponibilité d'une page, procédez comme suit:

REMARQUE: Vos modifications seront automatiquement enregistrées, il n'y a pas de bouton d'enregistrement ou de confirmation sur lequel cliquer.
  1. Ouvrez le module Layout.
  2. Cliquez sur le nom de la page pour la modifier. Les détails de la page s'affichent.
  3. À droite de Disponibilité, cliquez sur .
  4. La boîte de dialogue Paramètres de la page s'ouvre, dans laquelle vous pouvez définir la disponibilité de la page.
  5. Une fois que vous avez configuré les paramètres de disponibilité, ils s'affichent dans la marge de gauche.

Configuration des restrictions du calendrier

Par défaut, lorsqu'une page est créée, elle sera toujours disponible pour consultation. Les pages peuvent être configurées avec des restrictions de calendrier qui définissent la date et l'heure de début auxquelles une page sera disponible pour consultation. Vous pouvez également configurer une date et une heure de fin. Pour configurer la planification d'une page, procédez comme suit:

  1. Revenez à la boîte de dialogue Paramètres de page et à la section Disponibilité.
  2. Pour les restrictions du calendrier, activez le bouton à bascule. Les paramètres de restriction du calendrier s'affichent.
  3. Cliquez sur l'icône du calendrier () et choisissez une date de début.
  4. Cliquez sur l'icône de l'horloge () et choisissez une heure de début.
  5. (Facultatif) Cochez la case Date de fin et sélectionnez une date et une heure de fin.
  6. Une fois que vous avez programmé la disponibilité d'une page publiée pour une date ultérieure, l'étiquette d'état de la page affiche Planifié.

Configuration des restrictions d'appareils

Par défaut, lorsqu'une page est créée, elle peut être consultée sur tous les appareils. Les pages peuvent être configurées de manière à ne pouvoir être consultées que sur des appareils spécifiques. Pour configurer les appareils sur lesquels une page peut être consultée, procédez comme suit:

  1. Revenez à la boîte de dialogue Paramètres de page et à la section Disponibilité.
  2. Pour Restrictions relatives aux appareils, activez le bouton à bascule. Les paramètres de restriction de l'appareil s'affichent.
  3. Sélectionnez/désélectionnez les appareils pouvant afficher la page.

Configuration des restrictions d'emplacement

Par défaut, lorsqu'une page est créée, elle peut être consultée dans tous les emplacements géographiques. Les pages peuvent être configurées de telle sorte que la consultation soit autorisée ou refusée dans des emplacements spécifiques. Pour configurer les emplacements d'une page, procédez comme suit:

  1. Revenez à la boîte de dialogue Paramètres de page et à la section Disponibilité.
  2. Pour Restrictions d'emplacement, activez le bouton à bascule. Les paramètres de localisation s'affichent.
  3. Dans la section Emplacements autorisés, tapez le nom d'un pays. Au début de la saisie, la liste sera filtrée.
  5. Cliquez sur un pays pour le sélectionner.
  6. Dans la section Lieux refusés, cliquez sur + Restrictions d'emplacement et ajoutez les pays dans lesquels la page ne peut pas être consultée.

Pour supprimer un pays, cliquez sur le X en regard du pays.
