Fonctionnalités/informations disponibles

Cette page d'accueil du module Layout fournit les fonctionnalités/informations suivantes:

Actualiser la mémoire cache

Cela actualise le cache de l'application afin que vous puissiez voir vos modifications immédiatement. Ce processus ne doit pas être effectué sans discernement, car le faire trop fréquemment en production a un effet négatif sur l'évolutivité et les performances.

Paramètres de l'application

Cliquez sur l'icône en forme d'engrenage pour ouvrir la page Paramètres de l'application.

Il s'agit de la page Paramètres de l'application qui apparaît lorsque vous cliquez sur l'icône représentant une roue dentée

Listing des pages

Chaque page créée est répertoriée dans la liste des pages. Voici les détails de la liste:

  • Poignée cliquer-déplacer: Icône indiquant que la page peut être déplacée par cliquer-glisser. Vous pouvez cliquer n'importe où sur le nom de la page pour la déplacer.
  • Indicateur d'état de la page: Indique l'état de la page, comme suit:
    • : En direct
    • : Non disponible sur le Web
    • : Non publié
    • : Mise en ligne programmée
  • Titre de la page: Titre de la page. Au survol, vous voyez le nom gris sous le titre (825-DK-Home- (2)) est le nom système de la page. Si la page est publiée, le titre de la page sera en noir, s'il n'a pas été publié en gris clair.
  • Indicateur de disponibilité des pages: S'il est affiché, indique que les paramètres de disponibilité empêchent actuellement l'affichage de la page dans les applications. Les informations contextuelles indiquent les restrictions qui empêchent la disponibilité, dans ce cas Restrictions d'appareils. La fenêtre contextuelle contient également un lien vers la page où vous pouvez modifier les restrictions de disponibilité.
  • Bouton Dupliquer: Cliquez pour créer une copie de la page. Le nom en double sera au format Nom de la page (copie).
  • Bouton de dialogue Paramètres de page: Cliquez pour ouvrir la boîte de dialogue des paramètres de page. Ici, vous pouvez modifier le titre de la page, les paramètres de disponibilité et archiver la page.

Créer une page

Cliquez sur Créer une page pour démarrer le processus de création de la page.

Supprimer ou restaurer des pages archivées

Cliquez sur Pages archivées pour afficher la liste de toutes les pages archivées à partir desquelles vous pouvez choisir de supprimer ou de restaurer la page archivée.
