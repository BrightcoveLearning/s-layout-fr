Introduction

If you have a page with a carousel that contains a playlist of more than 6 or 10 (depending on platform) videos you can create a linked page that will automatically be shown in lieu of the very wide carousel. Normally you would want the large number of videos display in a format, like a grid, where the viewer doesn't have to scroll excessively to see all the videos.

Note that a linked page can only be defined for a page that displays videos in a carousel. The single exception to this statement is a linked page cannot be created when the Hero (Autoplay Channel) is the layout selected for the page from which you wish to link.

Note: For clarity, in the steps below the term main page will refer to the page where you choose the playlist which will display the linked page. On the main page you will see a Linked page indicator (highlighted) associated with the playlist added to the main page.

Creating a linked page

The following are the steps required to create a linked page:

  1. In the Layout module navigate to the properties of a page that contains a carousel.
  2. To the right of the playlist whose contents you wish to display in a linked page, click the ellipses (...).
  3. In the playlist settings dialog which opens, click More layout options.
  4. In the playlist layout dialog which opens, to the right of Linked page, click Add.
  5. Choose the layout for the linked page.
  6. Click Create.
  7. Confirm you are now viewing the properties for the linked page.
  8. Click Publish and confirm your action in the dialog that appears.
  9. Return to your main page's properties.
  10. Check to be sure you can see the Linked page indicator (highlighted).

Edit a linked page

To edit an existing linked page, click on the Linked page indicator associated with the playlist added to a main page.

You will then see the linked page's details.

Delete a linked page

To delete an existing linked page, click on the Linked page indicator associated with the playlist added to the main page.

You will then see the linked page's details, where you can click Delete.

Access linked pages from the page list

So far in the document you have accessed details about your linked page by clicking on the Linked page indicator associated with the playlist added to the main page. You can also access linked pages from the index page of the Layout module. To do so, follow these steps:

  1. From the index page of the Layout module, scroll to the bottom of the page list until you see the Linked pages text, which is followed by the number of linked pages in your experience.
  2. Click Linked pages.
  3. Check to be sure you see a list of all the linked pages you have created.
  4. Click one one of the links and you will see the linked page's details.
  5. From here you can edit, delete or publish your linked page. Note that you cannot set availability on linked pages.
