Présentation

Lorsque vous apportez des modifications à votre application {{site.product}}, vous souhaiterez souvent voir ces modifications dès que possible. Pour accélérer le processus, vous pouvez vider le cache afin d'actualiser vos applications et de voir les modifications. Cela réduira le délai avant que vos modifications n'apparaissent dans vos applications.
