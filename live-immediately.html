Présentation

Lorsque vous apportez des mises à jour à une page du module Mise en page, ces mises à jour sont immédiatement mises en ligne. Vous verrez un message dans le module Mise en page lorsque vous allez modifier une page indiquant ceci:

Remarque: Le lien En savoir plus dans le message permet d'ouvrir ce document.

Création d'une révision

Pour le moment, il n'existe pas de méthode automatique pour créer une révision d'une page, mais vous pouvez le faire manuellement. Suivez ces étapes:

  1. Dupliquez la page actuelle: Pour ce faire, revenez à la page d'index de la mise en page. Dans la liste des pages, cliquez sur l'icône de duplication en regard de la page pour laquelle vous apportez une révision.
  2. Apportez des modifications à la page dupliquée: Cliquez sur la page en double pour accéder à sa page de détails. Utilisez les outils disponibles pour mettre à jour la page comme vous le souhaitez.
  3. Publiez la page en double: La nouvelle page apparaîtra alors dans le menu.
  4. Dépubliez l'ancienne page description: Cette page n'est plus à jour, vous devez donc la dépublier, ce qui la supprime du menu.

Vos modifications seront désormais prises en compte dans la nouvelle page.
