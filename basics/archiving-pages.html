L'archivage d'une page permet de la dépublier et de la supprimer de l'application. Les pages archivées peuvent être supprimées ou restaurées dans l'application. Pour archiver une page, procédez comme suit:

  1. Ouvrez le module Layout.
  2. Survolez le nom d'une page, puis cliquez sur à côté. La boîte de dialogue Paramètres de page s'ouvre.
  3. En bas à gauche de la boîte de dialogue des paramètres de page, cliquez sur Archiver la page.
  4. Notez que la page est immédiatement archivée et que vous revenez à la page d'index du module Disposition alors que la page archivée ne figure plus dans la liste des pages.

Pour afficher toutes les pages archivées et les restaurer ou les supprimer définitivement, procédez comme suit:

  1. Tout en bas du panneau de navigation du module Mise en page, cliquez sur le lien qui indique le nombre de pages archivées:
  2. Vérifiez que la liste des pages archivées s'affiche:
  3. Passez la souris sur le nom de la page que vous souhaitez restaurer ou supprimer.
  4. Cliquez pour restaurer la page archivée.
  5. Cliquez pour supprimer définitivement la page. La boîte de dialogue de confirmation Supprimer la page s'affiche.
  6. Revenez à la liste Disposition de toutes les pages. Si vous avez restauré une page, elle apparaîtra dans la liste des pages, mais dans un état non publié.
