Creating a revision

At this time there is not automatic way to create a revision of a page, but you can do so manually. Follow these steps:

  1. Duplicate the current page: This is done by returning to the Layout index page. In the page list, click the duplicate icon next to the page for which you are making a revision.
  2. Make changes to the duplicate page: Click on the duplicate page to enter its details page. Use the available tools to update the page as wished.
  3. Publish the duplicate page: This will make the new page appear in the menu.
  4. Unpublish the older page: This page is now out of date so you will want to unpublish it, which removes it from the menu.

Your changes will now be reflected in the new page.
