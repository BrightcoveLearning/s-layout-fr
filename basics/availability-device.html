Vous pouvez configurer les pages pour qu'elles apparaissent uniquement sur des appareils spécifiques. Pour configurer les appareils autorisés/refusés pour une page, procédez comme suit:

Lorsque vous apportez des modifications aux paramètres, un cercle vert avec une coche s'anime pour indiquer que vos modifications sont automatiquement enregistrées.
  1. Ouvrez le module Layout.
  2. Dans le panneau de navigation latéral, survolez le nom de la page, puis cliquez sur .
  3. Dans la boîte de dialogue Paramètres de page, cliquez sur Disponibilité.
  4. À l'extrême droite de la ligne Restrictions relatives aux appareils, cliquez sur le bouton bascule.
  5. Cochez les cases pour autoriser l'affichage d'une page à partir d'un groupe ou d'un appareil individuel.
Vous pouvez également configurer les restrictions relatives aux appareils en cliquant sur le nom d'une page. Dans les détails de la page, dans la section Disponibilité, cliquez sur .
