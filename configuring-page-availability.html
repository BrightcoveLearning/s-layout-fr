Page availability settings can be used to control:

  • The dates a page is available
  • The device(s) a page is available on which to be viewed
  • The locations where a page can be viewed

To configure the Availability settings for a page, follow these steps:

NOTE: Your changes will be automatically saved, there is no save or confirm button to click.
  1. Open the Layout module.
  2. Click on the page name to edit. The page details will be displayed.
  3. To the right of Availability, click .
  4. The Page Settings dialog will open, where you can set availability for the page.
  5. After you configure Availability settings, they will be displayed in the left margin.

Configuring calendar restrictions

By default, when a page is created, it will always be available for viewing. Pages can be configured with calendar restrictions that define the start date and time that a page will be available for viewing. Optionally, you can also configure an end date and time. To configure a schedule for a page, follow these steps:

  1. Return to the Page Settings dialog and the Availability section.
  2. For Calendar restrictions, turn on the toggle. Calendar restriction settings will display.
  3. Click the calendar icon () and pick a start date.
  4. Click the clock icon () and pick a start time.
  5. (Optional) Check the End Date checkbox and select an end date and time.
  6. After you schedule a published page to be available for a future date, the page status label will display Scheduled.

Configuring device restrictions

By default, when a page is created, it will be available for viewing on all devices. Pages can be configured so they can only be viewed on specific devices. To configure the devices a page can be viewed on, follow these steps:

  1. Return to the Page Settings dialog and the Availability section.
  2. For Device restrictions, turn on the toggle. Device restriction settings will display.
  3. Select/Unselect the devices that can view the page.

Configuring location restrictions

By default, when a page is created, it will be available for viewing in all geographic locations. Pages can be configured so viewing is permitted or denied in specific locations. To configure the locations for a page, follow these steps:

  1. Return to the Page Settings dialog and the Availability section.
  2. For Location restrictions, turn on the toggle. The location settings will display.
  3. In the Permitted Locations section, type the name of a country. As you begin typing, the list will filter.
  5. Click on a country to select it.
  6. In the Denied Locations section, click + Location restrictions and add the countries where the page cannot be viewed.

To remove a country, click the X next to the country.
