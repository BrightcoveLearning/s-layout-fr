Vous pouvez configurer les pages pour qu'elles n'apparaissent que dans des emplacements spécifiques. Pour configurer les emplacements autorisés/refusés, procédez comme suit:

Lorsque vous apportez des modifications aux paramètres, un cercle vert avec une coche s'anime pour indiquer que vos modifications sont automatiquement enregistrées.
  1. Ouvrez le module Layout.
  2. Dans le panneau de navigation latéral, survolez le nom d'une page, puis cliquez sur .
  3. Dans la boîte de dialogue Paramètres de page, cliquez sur Disponibilité.
  4. À l'extrême droite de la ligne Restrictions d'emplacement, cliquez sur le bouton bascule.
  5. Sélectionnez les emplacements pour les emplacements autorisés et refusés souhaités en saisissant le nom d'un pays/d'un lieu. Au début de la saisie, la liste sera filtrée.
Vous pouvez également modifier le calendrier en cliquant sur le nom d'une page. Dans les détails de la page, dans la section Disponibilité, cliquez sur .
